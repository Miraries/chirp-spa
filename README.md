# Chirp SPA

A Twitter clone using Laravel and Vue.js, Vuex, styled with Bootstrap

## Getting Started

### Installing - Development

```
composer install
npm run dev
php artisan migrate
```

Enter data into database (Factories may not be set-up) before running server:

```
php artisan serve
```
